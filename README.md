# Stopwatch Web Application

This project is a simple stopwatch built using HTML, CSS, and JavaScript.

## Features
- Start the stopwatch
- Pause the timer
- Reset the stopwatch
- Millisecond precision
- Clean and responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Click **Start** to begin the stopwatch.
2. Click **Pause** to stop the timer.
3. Click **Reset** to clear the timer.
